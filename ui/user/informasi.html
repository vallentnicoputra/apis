Cara Menggunakan API (cURL)

Gunakan format berikut untuk memanggil API kami melalui terminal atau aplikasi lain menggunakan cURL. Ganti `[path]` dengan endpoint yang Anda tuju dan `[API_KEY_ANDA]` dengan API Key Anda.

curl -X GET "https://Maslent.site/[path]&apikey=[API_KEY_ANDA]"

Informasi Penting

Pemberitahuan: Jika Anda melihat kotak respons kosong seperti pada gambar di atas setelah menekan "Jalankan Permintaan", itu **bukanlah error**. Server sedang dalam proses mengonversi hasil menjadi gambar (base64). Mohon tunggu beberapa saat hingga hasilnya muncul.
